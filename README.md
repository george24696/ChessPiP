# ChessPiP

A Chrome extension that lets you pop the Chess.com board out into a Picture-in-Picture window, so you can keep an eye on your game while working in other tabs or apps.

## Features

- Pop the live Chess.com board into a floating, always-on-top PiP window
- Keep watching your game (or a stream/analysis board) while you multitask
- One-click toggle from the extension icon

## Status

Early development — nothing to install yet. Watch this repo for the first release.

## Development

1. Clone the repo
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode**
4. Click **Load unpacked** and select the project folder
